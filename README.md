# tm

[![CI](https://github.com/winebarrel/tm/actions/workflows/ci.yml/badge.svg)](https://github.com/winebarrel/tm/actions/workflows/ci.yml)

Time Calculator.

## Usage

```
Usage: tm [<expr>]

Arguments:
  [<expr>]    Expression to evaluate.

Flags:
  -h, --help       Show help.
      --version
```

## Installation

```sh
brew install winebarrel/tm/tm
```
